# Proyecto_Arduino_App_Bluetooht
El concepto principal de este proyecto consiste en el control con arduino de cualquier aparato eléctrico a 230 V AC del entorno doméstico. 
Se trata de un dispositivo que conectado a la red AC alimenta el sistema de arduino a 5V CC, 
éste está compuesto por módulo un bluetooth HC05 y un relay, que a su vez es activado y 
desactivado desde la interface de una aplicación móvil, y que nos servirá para encender 
o apagar electrodomésticos a corta distancia (7 m.) lo que aportará al usuario comodidad en ciertas acciones.
